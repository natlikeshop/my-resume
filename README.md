# เว็บไซต์ Resume ของ นัฐพงษ์ นาคอ่วม (Nattaphong Nakuam)

เว็บไซต์ Resume ส่วนตัวสไตล์ Modern Minimalist ที่ออกแบบมาให้ดูพรีเมียมและใช้งานง่าย พร้อมฟีเจอร์ที่ทันสมัย

## 🌟 คุณสมบัติเด่น (Features)

- **ดีไซน์ระดับพรีเมียม:** ใช้โทนสีขาว-ม่วง และการไล่ระดับสี (Gradients) ที่ดูทันสมัย
- **Dark Mode:** รองรับโหมดมืดและโหมดสว่าง (Toggle Dark/Light Mode)
- **เอฟเฟกต์การพิมพ์ (Typing Effect):** ส่วนหัวของหน้าเว็บมีการพิมพ์ข้อความแสดงตำแหน่งงานแบบเคลื่อนไหว
- **การเคลื่อนไหวเมื่อเลื่อนหน้าจอ (Scroll Reveal):** เนื้อหาจะค่อยๆ ปรากฏขึ้นเมื่อเลื่อนหน้าจอลงมา
- **3D Card Effect:** การ์ดแสดงผลงานและประสบการณ์มีลูกเล่น 3D เมื่อนำเมาส์ไปวาง (Hover)
- **Responsive Design:** แสดงผลได้สวยงามทั้งบนคอมพิวเตอร์ แท็บเล็ต และมือถือ
- **Animated Background:** พื้นหลังแบบ Mesh Gradient ที่เคลื่อนไหวได้อย่างนุ่มนวล

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

- **HTML5:** โครงสร้างหลักของเว็บไซต์
- **Tailwind CSS:** การจัดการสไตล์และความสวยงาม (รวมถึง Dark Mode)
- **Lucide Icons:** ไอคอนที่ทันสมัยและเรียบง่าย
- **JavaScript (Vanilla JS):** จัดการตรรกะของโหมดมืด, เอฟเฟกต์การพิมพ์ และแอนิเมชันต่างๆ

## 🚀 วิธีการใช้งาน (Getting Started)

1. ดาวน์โหลดหรือ Clone โปรเจกต์นี้ลงในเครื่องของคุณ
2. เปิดไฟล์ `index.html` ผ่านเว็บเบราว์เซอร์ (เช่น Chrome, Firefox หรือ Safari)
3. หากต้องการพัฒนาต่อ คุณสามารถแก้ไขเนื้อหาใน `index.html` ได้โดยตรง
